ATAK Network Tool

Versiyon: 2026-PRO
Geliştirici: Presta Devolopment

📦 Hakkında

ATAK Network Tool, ağ ve IP yönetimi için geliştirilmiş bir Windows uygulamasıdır.
Kurulum sihirbazı ve ürün anahtarı sistemi ile birlikte gelir. Kurulum sırasında lisans doğrulaması yapılır ve kullanıcıya basit bir GUI deneyimi sunar.

Uygulama, kullanıcı dostu Tkinter arayüzü ile tasarlanmıştır ve tam ekran hacker havası vermektedir.

⚡ Özellikler

Kendi IP Adresini Görüntüleme

Site IP Bulma (Örn: google.com)

WiFi Adı Görüntüleme

İnternet IP Adresini Görüntüleme

Tam Ekran Kullanıcı Arayüzü

Hacker Temalı Tasarım

Kurulum Sihirbazı ve Ürün Anahtarı Doğrulama


🔑 Ürün Anahtarı

Kurulum sırasında ürün anahtarı girilmesi gereklidir.

Eğer ürün anahtarınız yoksa:

“Ürün Anahtarım Yok” seçeneğine basın.

Yeni ürün anahtarı almak için e-posta gönderin: ubuntikadam@gmail.com

💻 Kurulum

Setup.exe dosyasını çalıştırın.

Kurulum sihirbazındaki adımları takip edin:

Sözleşmeyi kabul edin

Kurulum dizinini seçin

Kurulum ilerleme çubuğunu takip edin

Ürün anahtarını girin veya yeni anahtar talep edin.

Kurulum tamamlandıktan sonra ATAK Network Tool otomatik olarak açılır.

🛠 Gereksinimler

Windows 10 veya üstü

Python 3.14 (sadece kaynak kod ile çalıştırmak isteyenler için)

PyInstaller ile derlenmiş .exe dosyası

💡 Kullanım

Program açıldığında tam ekran ve hacker temalı arayüz sizi karşılar.

Butonlar ile kendi IP’nizi, internet IP’nizi, bağlı WiFi adını veya herhangi bir sitenin IP’sini görebilirsiniz.

Çıkış için “Çıkış” butonunu kullanın.
